Fibonacci Sequence Generator

About

This website generates a Fibonacci Sequence to a specified number of iterations.

The sequence is generated by a recursive function as seen below:

      const fibRec = (i sequence= []) =>
        i === 0
          ? sequence
          : sequence.length === 0
          ? fibRec(i - 1, [0])
          : sequence.length === 1
          ? fibRec(i - 1, [0, 1])
          : fibRec(i - 1, [
              ...sequence,
              sequence[sequence.length - 1]
               + sequence[sequence.length - 2],
            ]);
